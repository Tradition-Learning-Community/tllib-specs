# Revue scientifique semi-automatique TLC

Ce répertoire contient uniquement des adjudications **candidates**. Il ne modifie ni les décisions humaines existantes, ni les extractions, ni les registres scientifiques.

Les entrées sont les 12 paquets de `reports/scientific-review/batches`, l'index global des objets, les candidats de doublons, les 496 signaux de couverture et les 830 termes non résolus. `decision-classification.yaml` constitue la vue exhaustive des 147 décisions. `automatic-decisions.yaml` contient seulement les actions admissibles par la politique et `human-exceptions.yaml` seulement les cas exigeant encore une personne.

La reproduction et la validation sont documentées dans `tools/scientific-auto-review/README.md`.
