# Dépendances Dynamics vers Huit Dimensions

Huit Dimensions n'est pas canonique dans `origin/main`. Dynamics utilise des grandeurs pouvant être rapprochées de dimensions, mais aucune identité n'est explicitement établie.

La dépendance demeure candidate/unresolved. Les contrats structurels peuvent être préparés sans elle; toute interprétation exécutable exige une future réconciliation.
