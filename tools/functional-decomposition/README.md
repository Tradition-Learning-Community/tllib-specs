# Functional decomposition tools

These deterministic tools operate only on the authorized snapshot and generated candidate registries.

- `python tools/functional-decomposition/validate_features.py` validates identity, traceability, blockers, readiness, references, variants, dependencies, YAML readability, and count consistency.
- `python tools/functional-decomposition/summarize_features.py` prints the registry counts and main feature categories.

Neither tool approves scientific content or generates mathematical contracts, IR, implementation code, bindings, backends, or public APIs.
